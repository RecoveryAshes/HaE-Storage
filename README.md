## 项目介绍

**HaE** 是一款面向**网络安全（数据安全）领域**的框架式项目，采用**乐高积木式**模块化设计理念，实现对本地文件与 HTTP 消息（包含 WebSocket）的精细化标记和提取。

GitHub 项目地址：https://github.com/gh0stkey/HaE  
GitCode 项目地址：https://gitcode.com/gh0stkey/HaE
